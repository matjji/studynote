### 5. 절대경로와 상대경로 개념
    절대 경로는 상위폴더에서 하위 폴더로 가는 경우이다.    
    ex) C:\Users\Sin_jisup\a\b\c    
    이런식으로 완전한 파일 경로를 입력하면 된다.

    상대 경로   
    ex) cd ..\a\b\c
    이런식으로 완전한 파일 경로가 아닌 몇부분을 생략해서 입력하면 된다.
    cd .. 입력시 상위폴더 갈 수 있다.
